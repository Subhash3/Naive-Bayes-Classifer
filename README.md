### Dataset format
* A dataset is a list of datasample of the following interface
```ts
interface IFC_Iris_Data_Sample {
    features: number[],
    category: string
}

Eg:
dataset = [
    {
        "features": [
            3.393533211,
            2.331273381
        ],
        "category": 0
    },
    {
        "features": [
            3.110073483,
            1.781539638
        ],
        "category": 0
    }
]
```
